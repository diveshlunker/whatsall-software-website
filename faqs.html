  • Account

  • Can I login from multiple device?

    Yes, you can download the software in any device and use your credentials to sign in.

  • How do I get License Key?

    Contact customer care team to get License Key.

  • Can we get multiple id's registered for same License?

    Yes, you can request any number of id's registered for same License.

  • Is there any admin panel?

    Admin panel is in our upcoming features plan. Please check Upcoming Features page for more details.

  • Messaging

  • How to send customized messages?

    Please check out our Blog on customized messages or view our youtube tutorial to learn about sending customized messages.

  • How can I resend message to only few users of a group?

    You can select the users to whom message should be resent from dashboard page in whatsall software.

  • Can I send Images / Documents?

    Yes, images / documents / gifs can be added in messages groups and the same will be sent to the users in whatsapp.

  • Is there any file size limit?

    No, you can send files of any size via whatsall.

  • Troubleshooting

  • I forgot my passoword, how do I reset it?

    We will soon bring in feature to reset password. Until then please reach out to customer care to get your password changed.

  • Unknown failure shown in dashboard. How do I handle such failures?

    Unknown failure can occur due to multiple reasons like slow internet, whatsapp being logged out etc. Please retry messaging from dashboard again.

  • Getting error: "Id already logged in from another device"?

    In case you have not logged in from another device and yet the error is thrown, please restart the software.

  • Error showing "Invalid License Key"?

    If you have entered correct license key and still getting the error, try checking for any extra space before/after the key. \nIf the issue still persists, reach out to customer care.

  • Unable to click on input box in different forms?

    This issue is very unlikely to happen. However if this happens, click "Alt+Tab" and retry.

  • Payments

  • Can I have an invoice for my subscription?

    In case you did not receive invoice to your registered email / mobile number, reach out to customer care.

  • How do I extend my subscription?

    Please reach out to customer care to extend your subscription.
